== Description ==

This powerful WordPress plugin lets you extend the functionality of the default WordPress gallery. You can easily customize the look and feel of the photo or video gallery using the custom GT3 option panel. Create a new look for your galleries with ease.

**GT3 Photo & Video Gallery [Demo Page](http://bit.ly/1RWUs0k "Demo Page")**

[youtube http://www.youtube.com/watch?v=eIUfmr91D8g]

The advantages of the plugin:

* Intuitive and easy to use
* Using native WordPress gallery and its functionality
* The photo and video galleries are fully responsive
* Custom gallery options panel
* Set the number of columns 1-9
* Thumbnail gallery view
* Masonry gallery view
* Grid gallery view
* Circle gallery view
* Link the images to lightbox
* Youtube and Vimeo video support
* Custom CSS support
* And much more...

== Installation ==

After downloading the ZIP file,

1. Log in to the administrator panel.
2. Go to Plugins Add > New > Upload.
3. Click Choose file (Browse) and select the downloaded zip file of the gallery.
4. Click Install Now button.
5. Click Activate Plugin button for activating the gallery plugin.
If the installation does not succeed, please contact us.
